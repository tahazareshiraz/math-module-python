# taha-math-shiraz

یک کتابخانه‌ی ریاضی برای پایتون که می‌تونی به جای ماژول `math` استفاده کنی، با این تفاوت که امکانات خیلی بیشتری داره. همه‌ی توابعش از صفر نوشته شده و به هیچ کتابخانه‌ی دیگه‌ای (حتی خود `math`) وابسته نیست.

⚠️ این کتابخانه هنوز توی PyPI ثبت نشده، یعنی فعلاً با `pip install taha-math-shiraz` نصب نمی‌شه. باید فایل‌ها رو دستی دانلود کنی (روش زیر).

## نصب

فایل zip رو دانلود و باز کن، بعد داخل همون پوشه این دستور رو بزن:

```bash
pip install .
```

یا اگه نمی‌خوای نصبش کنی، کافیه پوشه‌ی `taha_math_shiraz` رو کنار پروژه‌ت کپی کنی و مستقیم importش کنی.

## نحوه استفاده

```python
import taha_math_shiraz as tms

tms.sqrt(16)          # رادیکال
tms.sin(tms.pi / 2)   # سینوس
tms.factorial(5)      # فاکتوریل
```

## چه کارهایی می‌تونه انجام بده؟

علاوه بر همه‌ی توابع معمول ریاضی (رادیکال، توان، لگاریتم، مثلثاتی، فاکتوریل، گذرپذیری، گرد کردن و...) این امکانات هم اضافه شده:

- **آمار و احتمال**: میانگین، واریانس، انحراف معیار، رگرسیون خطی، همبستگی، توزیع‌های آماری (نرمال، دوجمله‌ای، پواسون و...)
- **نظریه اعداد**: تشخیص اعداد اول، تجزیه به عوامل اول، مقسوم‌علیه‌ها، دنباله فیبوناچی
- **جبر خطی**: کار با ماتریس و بردار (ضرب، دترمینان، وارون کردن، حل دستگاه معادلات)
- **هندسه**: مساحت و محیط اشکال مختلف، فاصله‌ی بین نقاط، تقاطع خطوط
- **مالی**: محاسبه‌ی بهره مرکب، اقساط وام، ارزش فعلی و آینده‌ی پول
- **فیزیک**: فرمول‌های حرکت، انرژی، الکتریسیته
- **گراف**: پیدا کردن کوتاه‌ترین مسیر بین دو نقطه (Dijkstra و BFS)
- **مرتب‌سازی و جست‌وجو**: چند الگوریتم مرتب‌سازی معروف برای یادگیری
- یه عالمه ابزار کوچیک دیگه مثل تبدیل واحد، انیمیشن (easing)، اعداد رومی، رمزنگاری ساده و غیره
